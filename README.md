

## Descripción del proyecto

La aplicación representa un semáforo simple utilizando componentes de Jetpack Compose.  
El cambio de luces se realiza de forma automática mediante un ciclo infinito controlado por corrutinas, lo que permite simular el comportamiento real de un semáforo.

Se utiliza un enum class para definir los posibles estados del semáforo (Red, Yellow, Green) y un estado reactivo para actualizar la interfaz cuando el color activo cambia.
